CHERRIFT v0.5.0 FULL PATCH
==========================

INSTALL:
1. Extract this ZIP.
2. Copy the included src folder into the CHERRIFT project root.
3. Allow files to merge.
4. Replace the existing src/main.js when asked.
5. Keep all existing files, especially:
   - src/cherrift_v040.js
   - src/cherrift_v042_completion.js
6. Hard refresh the browser or clear site cache.

NEW FILES:
- src/cherrift_v050.js
- v050.css

REPLACED FILE:
- src/main.js

v0.5 FEATURES:
- Persistent Player/Account Level and XP
- Account stat bonuses
- World 3 with five stages
- New World 3 enemy family and boss
- Enemy and boss equipment drops
- Common, Uncommon, Rare, Epic and Legendary rarity
- Item level and item power
- Inventory capacity and auto-conversion when full
- Inventory filtering and sorting
- Item lock/unlock
- Equipped-item comparison
- Updated sell values
- Guaranteed boss loot
- World 3 unlock after World 2-5
- Save schema upgraded to v5

COMPATIBILITY:
This patch does not replace or edit cherrift_v040.js.
The World 2 FPS fixes remain active.
The v0.4.2 save/autosave patch loads before v0.5.

IMPORTANT:
World 3 currently uses the existing map artwork with a fire/ember visual treatment.
Dedicated World 3 images and sprite sheets can be added later without changing the progression system.
