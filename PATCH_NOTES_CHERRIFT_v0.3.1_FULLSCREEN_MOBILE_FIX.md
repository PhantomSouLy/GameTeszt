# CHERRIFT v0.3.1 Fullscreen Mobile Hotfix

## Fixed
- On mobile fullscreen, pressing Play from World Select could show only the Skill button while the canvas/stage stayed invisible until leaving fullscreen.
- Forced canvas viewport recalculation after stage start.
- Added delayed resize/render passes after fullscreen changes.
- Added fullscreen CSS for `#game` and `#app`.

## Preserved
- PC menu layout unchanged.
- World Select carousel unchanged.
- Progression / raid / reward systems unchanged.
