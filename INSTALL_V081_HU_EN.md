# CHERRIFT v0.8.1 telepítés

1. A ZIP tartalmát másold közvetlenül a CHERRIFT repository gyökerébe.
2. Engedélyezd a `src/main.js` felülírását.
3. Az új fájlok:
   - `src/cherrift_v081.js`
   - `v081.css`
4. Commit és push után várd meg a GitHub Pages frissülését.
5. Végezz hard refresh-t: `Ctrl + F5`.

Ajánlott commitüzenet:

```text
CHERRIFT v0.8.1 Arsenal layout hotfix
```

Ez a hotfix a v0.8.0 csomagra épül. A v0.7 és v0.8 fájlokat ne töröld.
