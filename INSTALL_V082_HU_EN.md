# CHERRIFT v0.8.2 telepítés / Installation

## Magyar

1. Készíts biztonsági mentést a jelenlegi repóról.
2. Csomagold ki a ZIP-et.
3. A ZIP **tartalmát** másold közvetlenül a CHERRIFT gyökerébe, ahol az `index.html` található.
4. Engedélyezd a `src/main.js` felülírását.
5. A következő új fájloknak kell megjelenniük:
   - `src/cherrift_v082.js`
   - `v082.css`
6. Commitold és pushold kézzel.
7. GitHub Pages frissülése után használj `Ctrl + F5` frissítést.

A csomag a jelenlegi v0.8.1 állapotra épül. A v0.7.0, v0.8.0 és v0.8.1 fájlokat ne töröld.

## English

1. Back up the current repository.
2. Extract the ZIP.
3. Copy the ZIP **contents** into the CHERRIFT repository root beside `index.html`.
4. Allow `src/main.js` to be replaced.
5. Confirm these new files exist:
   - `src/cherrift_v082.js`
   - `v082.css`
6. Commit and push manually.
7. After GitHub Pages deploys, hard-refresh with `Ctrl + F5`.

This package is built on top of v0.8.1. Keep the v0.7.0, v0.8.0 and v0.8.1 patch files.
