CHERRIFT v0.5.5.1 MOBILE HOTFIX

INSTALL
1. Copy all ZIP contents into the CHERRIFT project root.
2. Merge the src folder.
3. Replace src/main.js.
4. Keep all earlier files.
5. Hard refresh / clear browser cache.

FIXES
- Mobile Profile, Stats, Gear Collection, Enemies, Skins and Worlds are now
  inside one Library panel with top tabs.
- The old separate mobile Collection / Bestiary / Profile panels are hidden.
- The bottom Library button opens the combined Library.
- Mobile Gear disables browser drag and uses tap/click only.
- Gear buttons and item cards receive stronger pointer-event handling.
- The main mobile navigation is hidden during gameplay, pause, level-up,
  loading, game over and stage clear.
- Desktop menus and desktop Gear are unchanged.
