# CHERRIFT v0.3.1 World Select Carousel Hotfix

## Changed
- Main Play now opens World Select instead of starting instantly.
- Mobile Play also opens World Select.
- World Select is now a single centered carousel card.
- Stage switching works with left/right arrows and swipe/drag.
- Carousel always starts from World 1-1.
- First stage has no left navigation.
- Bottom actions: Back + Play.
- Play starts only from this World Select screen.
- Locked stages can be viewed, but cannot be launched.

## Preserved
- PC main menu layout stays unchanged.
- Existing progression, raid, reward and Beastclaw systems remain.
