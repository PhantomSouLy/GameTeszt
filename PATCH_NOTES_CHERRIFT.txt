CHERRIFT v0.3.4 Early Build Patch Notes

[Added]
- Pause / Settings polish update.
- New pause button: SETTINGS.
- You can now open Settings while a run is paused.
- Settings now has quick actions:
  - BACK / BACK TO PAUSE
  - RESUME RUN
- Added more settings controls:
  - UI scale
  - View zoom (More Map / Balanced / Closer)
  - Compact objective HUD toggle
  - Damage numbers toggle
  - Fullscreen toggle from Settings
- Added src/pause_settings_polish_v034.js

[Changed]
- Pause modal layout is cleaner and more readable.
- Settings layout is more mobile-friendly.
- Settings can now behave differently depending on where it was opened from:
  - main menu -> back returns to menu
  - pause menu -> back returns to pause
- UI scale now affects the HUD and skill button.
- View zoom now has a user-facing control to adjust how close the gameplay camera feels.

[Kept]
- CHERRIFT branding only.
- v0.3.3 combat/HUD/upgrade prep remains included.
- In-game fullscreen support remains included.
- World Select flow remains:
  Main Play -> World Select -> Play -> Loading -> Stage
