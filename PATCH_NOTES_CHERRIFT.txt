CHERRIFT v0.3.3 Early Build Patch Notes

[Added]
- Combat-feel patch file: src/combat_hud_upgrade_patch_v033.js
- Generalized upgrade system prep.
- New upgrades are not skin-specific. They adapt by combat style.
- Multi Strike:
  - Ranged skins: +1 projectile.
  - Melee skins: +18% melee range.
- Combat Arc:
  - Ranged skins: wider projectile spread.
  - Melee skins: wider melee cone.
- Skill Flow: lower skill cooldown.
- Thorn Aura: nearby enemies take slow aura damage.
- Damage numbers on enemy hit.
- Death pop effect on enemy kill.
- Raid warning banner.
- Mini boss HP bar.
- Stage Clear screen polish.
- World Select button added to Stage Clear actions.

[Changed]
- HUD cleaned up.
- Duplicate kill text next to the timer is hidden.
- Stage objective is now the main kill tracker: World x-y · kills/goal.
- Stage objective no longer uses the heavy glass background.
- Raid info no longer uses the small glass box; raid now appears as a clear center warning.
- Version display updated to v0.3.3 EARLY BUILD.

[Kept]
- CHERRIFT branding only.
- World Select flow: Main Play -> World Select -> Play -> Loading -> Stage.
- Loading freeze fix remains included.
- Level-up freeze fix remains included.
- In-game fullscreen button remains in Pause menu.
- Progression numbers from v0.3.2 remain included.

[Design Rule]
- Common/Rare/Epic skins use universal and combat-style upgrades only.
- Legendary+ skins can later add unique skin-specific upgrades.
- This prevents every skin from needing a separate upgrade tree too early.
